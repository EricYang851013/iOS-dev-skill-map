# ES6

## 1. 关于 ES6
ES6 泛指 ES5.1 版本以后的 ECMAScript 标准，包括 ES2015、ES2016、ES2017。2015 年开始标准委员会决定标准在每年六月正式发布一次，作为当年的正式版本。
