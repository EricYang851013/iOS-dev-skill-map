## 3. Flexbox

### 3.1. Yoga

[Yoga](https://facebook.github.io/yoga/) 是 Facebook 在 React Native 里引入的一种跨平台的基于 CSS 的布局引擎，它实现了 Flexbox 规范，完全遵守 W3C 的规范（Weex 中的 Layout 文件是 Yoga 的前身，是 Yoga 正式发布之前的版本，因此 RN 和 Weex 可以说采用了同一套布局方案）。
